
## Project Description


This FTOptix sample project demonstrates how to implement **Mutual Exclusion Screen Access** functionality. It provides an example of controlling access to specific screens so that only one user or session can access a particular screen at a time, preventing concurrent access conflicts in multi-user HMI environments.
